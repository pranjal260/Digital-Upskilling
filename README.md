# Digital Upskilling Platform

A backend-supported web platform where users can explore and enhance their skills through structured content.

## Features
- Backend routing using Node.js and Express
- Data storage and retrieval using MongoDB
- Handles application requests and backend workflows

## Tech Stack
- JavaScript
- Node.js
- Express.js
- MongoDB
- Git & GitHub

## What I Worked On
- Implemented backend routes and server-side logic
- Integrated MongoDB for data storage
- Worked on backend workflows and API structure

## How to Run
1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables if required
4. Run the server using `npm start`

## Learning Outcomes
- Backend development using Node.js and Express
- Working with NoSQL databases (MongoDB)
- Understanding backend architecture and API flow
